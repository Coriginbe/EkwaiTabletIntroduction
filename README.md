# 鲲翼平板说明

本文档基于个人了解和经验，对鲲翼平板作简单说明，并提供关闭管控的解决方案。


## 鲲翼（Ekwai）简介

> “鲲翼”为广州摩翼信息科技有限公司子品牌，主要定位打造智慧教育硬件系列产品。与“翼课网”是兄弟品牌。

[鲲翼品牌介绍页](https://cloud.ekwing.com/?act=ekwai)

“鲲翼平板”是广州摩翼信息科技有限公司（以下简称“摩翼”）联合华为技术有限公司（以下简称“华为”）推出的产品，由华为提供硬件，摩翼提供管控系统与教育资源。该系列产品实质是安装了定制系统（基于 Android）的华为平板企业定制机，通过预装相关组件对用户行为进行管控，从而定制成学习机。

[鲲翼产品介绍页](https://cloud.ekwing.com/?act=protea)

## 鲲翼系统

“鲲翼系统”本体是一个名为「ekwLauncher」的 Launcher 应用，「ekwLauncher」被设为默认 Launcher 与系统应用，无法卸载，并激活为设备管理器，无法解除激活。

    管理器被允许执行以下操作：

    - 删除所有数据
    - 设置密码规则
    - 监视屏幕解锁尝试次数
    - 锁屏
    - 设置锁屏密码的有效期
    - 设置存储设备加密
    - 停用相机
    - 设置管理 App 的权限
    - 设置蓝牙
    - 设置 USB
    - 设置存储访问（MicroSD 卡）
    - 设置设备管理

被设为系统应用的厂商（摩翼）应用还有「鲲翼应用市场」「翼课教师」「翼课堂教师端」「翼课学生HD」「翼课堂学生端」。

---

主屏幕底部 Dock 栏有「鲲翼」和「安卓」两个按钮，点击可切换对应界面。

在「鲲翼」界面可登录翼课网账号。「鲲翼」界面登录状态与「鲲翼应用市场」「翼课教师」「翼课堂教师端」「翼课学生HD」「翼课堂学生端」同步。  
登录学生账号，「鲲翼应用市场」「翼课学生HD」「翼课堂学生端」同步登录；  
登录教师账号，「鲲翼应用市场」「翼课教师」「翼课堂教师端」同步登录。

根据登录账号类型，系统采取不同的管控策略，「鲲翼应用市场」提供不同的 App 资源。

未登录状态下，采取默认管控策略，「鲲翼应用市场」提供通用的 App 资源。

    - 隐藏部分设置项
    - 禁止通过连续点击版本号启用开发者选项
    - 禁止启用 USB调试
    - 禁止恢复出厂设置（设置，工程菜单、Recovery）
    - 禁止用户安装系统固件
    - 禁止蓝牙传输文件
    - 禁止安装白名单以外 App
    - 禁止访问白名单以外域名
